# LabCommand Threat Intelligence

## What This Is
This repository tracks malicious IP addresses and associated indicators of compromise (IOCs) observed against internet facing infrastructures. All data is sourced from live telemetries collected by LabCommand's environments.

## Methodology
IOCs are extracted from WAF/Server logs generated by inbound traffic coming to LabCommand systems and triaged based on request patterns, payload signatures, and behavioral characteristics. Each indicator is reviewed and categorized by attack intent before being committed to this repository. Enrichment is performed manually using open source intelligence tools (OSINT) and threat intelligence platforms to validate and contextualize findings prior to publication. 

## Tools Used
- **VirusTotal** 
- **AbuseIPDB**
- **AlienVault OTX**
- **urlscan.io**
- **Shodan**
- **IPinfo (GeoIP/ASN lookup)** 


## IOC Categories
- **Scanners** — IP addresses observed conducting automated port, service, and/or vulnerability recon against exposed infrastructures. 
- **Scrapers** — IP addresses observed making high volume and/or systematic HTTP requests consistent with content harvesting or enumeration activity
- **Shell Hunters** — IP addresses observed probing for web shells, exposed admin panels, and/or remote code execution vectors via malformed or suspicious URI requests.
- **Brute Forcers** — IP addresses observed making repeated authentication attempts against login endpoints, SSH, and/or other credential-protected services.
